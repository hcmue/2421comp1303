# Buổi 04 (10/03/2025)

- Javascript cơ bản
  - Cách chèn javascript vô trang html
    Có 2 cách:
    - Chèn trực tiếp code js trong khối <script></script>
    ```
        <script>
          console.log('Hello World');
      </script>
    ```
    - Tạo script trong \*.js rồi chèn vô trang html:
    ```
    <script src="myscript.js"></script>
    ```
